Icon Generator for TheraSoul

This tool will help generate proper icon files for the PWA manifest.

Instructions:

  1. Open the logo.png file in an image editor
  2. Resize it to 192x192 pixels and save as icon-192x192.png
  3. Resize it to 512x512 pixels and save as icon-512x512.png
  4. Place both files in the public/ directory
